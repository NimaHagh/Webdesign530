Problem 2

I wrote a JavaScript function that determines whether a string is a palindrome by first removing spaces, punctuation, and converting all uppercase letters to lowercase. This ensures that the comparison is case-insensitive and only considers alphanumeric characters. The function then checks if the cleaned string reads the same forward and backward. Based on the result, I display a message indicating whether the input string is a palindrome or not.
